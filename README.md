# Interactive DateTime Checker

## Description
This project provides an interactive webpage that shows the current date and time when the user clicks a button. The button toggles between "Check" and "Restart," changing its text and color accordingly to enhance user interaction.

## Features
- Displays the current date in UK format.
- Displays the current time.
- Button toggles between "Check" and "Restart" states.
- Button color changes dynamically according to its state.
- Responsive and centered design using Flexbox.

## Installation / How to Run
To run this project, simply open the `index.html` file in any modern web browser. No additional installation is required.

## Usage
- Click the **Check** button to display the current date and time.
- After displaying, the button changes to **Restart** with a new color.
- Click **Restart** to reset the display and button to its initial state.

## Technologies Used
- HTML5
- CSS3 (Flexbox for layout)
- JavaScript (Vanilla ES6+)

## Project Structure
- `index.html` - The main HTML file.
- `style.css` - Contains styles and layout definitions.
- `script.js` - Contains JavaScript logic for button interaction and date/time display.
